# Creating Menus on LCD

## Level
Intermediate

## Time Estimate
35 min

## Overview
This module teaches you about creating menus on lcd. You'll gain hands-on experience with essential Arduino concepts and build practical skills that apply to real-world projects. Through step-by-step guidance, you'll understand both the theory and implementation of this important topic.

## Prerequisites
- LCD Display (16x2)
- Using Pushbuttons & Debouncing

## Learning Outcomes
- Understand the fundamentals of creating menus on lcd
- Wire and configure components correctly
- Write clean, functional Arduino code
- Debug common issues effectively
- Apply knowledge to custom projects
