# Robotics: Line Following Robot

## Level
Advanced

## Time Estimate
50 min

## Overview
This module teaches you about robotics: line following robot. You'll gain hands-on experience with essential Arduino concepts and build practical skills that apply to real-world projects. Through step-by-step guidance, you'll understand both the theory and implementation of this important topic.

## Prerequisites
- DC Motors + L298N Driver
- Light Sensors (LDR)

## Learning Outcomes
- Understand the fundamentals of robotics: line following robot
- Wire and configure components correctly
- Write clean, functional Arduino code
- Debug common issues effectively
- Apply knowledge to custom projects
