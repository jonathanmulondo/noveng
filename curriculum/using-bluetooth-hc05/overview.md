# Using Bluetooth HC-05

## Level
Intermediate

## Time Estimate
30 min

## Overview
This module teaches you about using bluetooth hc-05. You'll gain hands-on experience with essential Arduino concepts and build practical skills that apply to real-world projects. Through step-by-step guidance, you'll understand both the theory and implementation of this important topic.

## Prerequisites
- Serial Monitor Basics

## Learning Outcomes
- Understand the fundamentals of using bluetooth hc-05
- Wire and configure components correctly
- Write clean, functional Arduino code
- Debug common issues effectively
- Apply knowledge to custom projects
