# Reading Rotary Encoders

## Level
Intermediate

## Time Estimate
25 min

## Overview
This module teaches you about reading rotary encoders. You'll gain hands-on experience with essential Arduino concepts and build practical skills that apply to real-world projects. Through step-by-step guidance, you'll understand both the theory and implementation of this important topic.

## Prerequisites
- Reading Button Inputs

## Learning Outcomes
- Understand the fundamentals of reading rotary encoders
- Wire and configure components correctly
- Write clean, functional Arduino code
- Debug common issues effectively
- Apply knowledge to custom projects
