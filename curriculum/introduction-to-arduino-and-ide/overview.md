# Introduction to Arduino & IDE

## Level
Beginner

## Time Estimate
15 minutes

## Overview
This module introduces you to the Arduino platform, what it is, and why it's used by millions of makers worldwide. You'll learn about the Arduino Uno board, its components, and how to set up the Arduino IDE (Integrated Development Environment) on your computer. By the end, you'll understand the basic structure of an Arduino program and be ready to upload your first sketch to the board.

## Prerequisites
- None (This is the starting point!)

## Learning Outcomes
- Understand what Arduino is and what it can do
- Identify the main components of the Arduino Uno board
- Install and configure the Arduino IDE
- Learn the basic structure of an Arduino sketch (setup and loop functions)
- Successfully upload a test program to your Arduino
